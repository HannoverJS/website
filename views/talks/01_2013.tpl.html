Talks

Damien Klinnert — Become a node package maintainer

Node's official package manager npm already has more then 17.000 entries.

These packages enable node developers to build web applications faster than ever. It's time to give something back to this great community and to contribute your own package.

Damien Klinnert will show how to become a node package maintainer with a detailed introduction and a best practices talk.

Christoph Burgdorf — How to waste your time on building a mobile quartett game

While enjoying the awesome fronteers conference in Amsterdam the hannoverjs peeps came up with this crazy idea to build an quartett game with conference speakers.

Well, but how would you actually rate them? We didn't want to lose all our credit on the interwebs so we quickly dropped the idea.

However, Christoph started hacking and built quartettjs - a project agnostic game engine - that can be used to build any quartett game on top of it

He also went on and built a mobile UI with Sencha Touch. Both projects are far from beeing something serious but still deliver plenty of opportunities to base a talk on :)

This talk is like a surprise egg: framework design, unit testing, sencha touch may or may not be featured in this talk. Attend and figure it out :)

Pascal Precht — AngularJS insights

There are a lot of JavaScript frameworks out there for building web applications. AngularJS is powered by Google and has a lot of interesting secrets in it. Secrets you might want to know, when building your own web application with Angular.

Pascal spent some time of figuring out, how Angular internally works and will show you not only the anatomy of Angular itself, but also things to remember when dealing with some of Angular's components.

Learn what HTML would have been had it been designed for applications!
