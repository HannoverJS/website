Christoph Burgdorf - ninya.io - a search engine to make skilled people in tech more discoverable

ninya.io is all about making skilled people in tech more discoverable. It might be best described as a search engine to find tech people. Basically it let's you search for people by their skillset and location. For instance here are some people from Hannover who seem to be web folks!

Christoph started ninya.io as a quick holiday hack in August 2013. Since then it has grown into a serious project with it's own set of challenges and fun parts.

In his talk Christoph will talk about the tech stack, including it's turbulent history, challenges and future plans.

As a bonus you might see some Go code in this talk!

Robin Thrift - Metalsmith - A fun, pluggable JavaScript static site generator

Imagine Jekyll mixed with Gulp and a little bit of magic. Add to that a dead simple plugin system and you've got yourself a pretty darn amazing static site generator. That's what the folks over at Segment.io thought to themselves and built Metalsmith, a pluggable JavaScript static site generator.

This talk will show you, exactly how easy it is, to build static websites using Metalsmith. From the first parsed markdown file to building a super simple plugin, you'll fall in love with Metalsmiths simplicity and its declarative nature.
